# Studio Todo

A Vite + Vue 3 todo app styled after the provided d.school-inspired layout.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Features

- LocalStorage-backed todo list
- Search, status filters, and completed-task cleanup
- D.school-inspired grid layout with bold editorial styling
- Responsive cards and task stats
